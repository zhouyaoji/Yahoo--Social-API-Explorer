Updates for Connections

Description

You can use this URI to get the most recent updates for the connections (friends) of the user specified by {guid}. The information returned includes profile data about the connections.

For more information, see Updates for Connections in the Yahoo! Social API Reference.

URI

GET returns a collection of updates for the connections of the profile owner with the given {guid}.

http://social.yahooapis.com/v1/user/{guid}/updates/connections
