This API explorer lets you make HTTP GET calls to the Yahoo! Social APIs by clicking links. The request and response headers, the returned response bodies (XML/JSON), and the related documentation.

Getting Started

  • Learn about the APIs: Mouse over the API names to see a summary of each API.
  • What is a GUID? Mouse over GUID for the answer.
  • Get Your GUID: Click the Introspective GUID link and find your GUID in the response body (XML/JSON).
  • Get Your Profile: Click Tinyusercard (most basic info) and Profile (complete profile info).
  • Understand the URI Syntax: Look at the URL in the URI text field. The syntax should match the URL below:
    • http://social.yahooapis.com/v1/user/{guid}/profile
  • Pagination: Click Connections: pagination. Pagination is done with the matrix parameters start and count.
  • Individual v. Collection Resources: Click Individual Connection and read about the differences between individual and collection resources.
  • Singleton URI and Response:Click one of the singleton resource URIs of the Connections API and look at the URI syntax and response.
    • http://social.yahooapis.com/v1/user/{guid}/connection/{cid}
  • Error Headers and Responses: Modify the {cid} in the URI for the individual connection and click Make Request.

    Take a look at the error response in XML/JSON. Also, note the HTTP Status Code in the response header.

  • Yahoo! Social API Documentation: The About tabs give a brief overview of the APIs. See the Yahoo! Social API Reference page by clicking on the breadcrumb link Documentation.
