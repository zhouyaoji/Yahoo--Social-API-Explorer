This API explorer lets you make HTTP GET calls to the Yahoo! Social APIs by clicking links. After you have clicked a link, the request and response headers, the returned response body in both XML and JSON, and the related documentation will appear on tabs. The XML and JSON tabs contain the response body and the About tab contains relevant documentation. The About the API Explorer tab moves to the far right, allowing you to complete Getting Started with the API Explorer.

Getting Started with the API Explorer

  1. 1. Learn about the APIs
    Mouse over Profiles API, Connections API, Contacts API, Status API, and Updates API to see a summary of each API.
  2. 2. What is a GUID?
    Mouse over GUID for the short answer. See Globally Unique Identifier (GUID) for a detailed explanation.
  3. 3. Get Your GUID
    From the Profiles API column, click "Introspective GUID" and find your GUID in the response body (XML/JSON). Your GUID is used for all other calls to the APIs and notated as {guid} in the documentation.
  4. 4. Get Your Profile
    Click "Tinyusercard" for basic profile info and then "Profile" for the complete profile.
  5. 5. Understand the URI Syntax
    Notice the URL in the URI text field. The following syntax is used for the collection resources: http://social.yahooapis.com/v1/user/{guid}/{resourcepath}
  6. 6. Pagination
    From the Connections API column, click "Connections: pagination". Pagination is done with the matrix parameters start and count that are appended to the base URI.
  7. 7. Individual v. Collection Resources
    Click the tab Individual Connection and read about the differences between individual and collection resources.
  8. 8. Singleton URI and Response
    From Individual Connection, click one of the singleton resource URIs of the Connections API and look at the URI syntax and response.
  9. 9. Error Headers and Responses:
    In the URI text field, modify the string at the end of the URL (/connection/{cid}) and click Make Request.

    Take a look at the error response on the XML and JSON tabs. Also, note the HTTP Status Code on the Response tab.

  10. 10. Yahoo! Social API Documentation
    The About tabs give a brief overview of the APIs. See the Yahoo! Social API Reference for more information.
