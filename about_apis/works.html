Work Profile Collection

Description

A Work Profile Collection is a collection of work profiles consisting of a user's stated work history.

For more information, see Work Profile Collection in the Yahoo! Social API Reference.

URI

GET returns a collection of work profiles of the profile owner specified by the given {guid}.

http://social.yahooapis.com/v1/user/{guid}/profile/works

Filters

Not available.

HTTP Operations Supported

  • GET

Query Parameters

  • callback
  • format

Matrix Parameters:

  • start
  • count

Scopes Required

  • GET:Read Yahoo! Profiles
