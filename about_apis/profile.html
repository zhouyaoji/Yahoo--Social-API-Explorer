Profile

Description

The profile contains a rich set of a user's social data, including school and work information as well as interests and relationships. As with all user data, information available in the profile will be subject to user permissions.

For more information, see Profiles in the Yahoo! Social API Reference.

URI

GET returns the profile of profile owner with the given {guid}.

http://social.yahooapis.com/v1/user/{guid}/profile

Filters

Calling GET on the following URI returns the profiles of the users identified by GUIDs {guid1}, {guid2}, and {guid3}.

/v1/users.guid(144,976,541)/profile

HTTP Operations Supported

  • GET

Query Parameters

  • callback
  • format

Matrix Parameters:

  • start
  • count

Scopes Required

  • GET:Read Yahoo! Profiles
