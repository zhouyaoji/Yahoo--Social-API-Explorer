School Profile Collection

Description

A School Profile Collection is a collection of school profiles entered by the profile owner.

For more information, see School Profile Collection in the Yahoo! Social API Reference.

URI

GET returns a collection of school profiles of the profile owner specified by the given {guid}.

http://social.yahooapis.com/v1/user/{guid}/profile/schools

Filters

Not available.

HTTP Operations Supported

  • GET

Query Parameters

  • callback
  • format

Matrix Parameters:

  • start
  • count

Scopes Required

  • GET:Read Yahoo! Profiles
