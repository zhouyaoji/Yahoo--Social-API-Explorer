Image

Description

Image returns the URL of an image associated with a particular user.

Image sizes: 16x16, 24x24, 32x32, 48x48, 64x64, 96x96, 128x128, 192x192

For more information, see Image in the Yahoo! Social API Reference.

URI

GET returns an image with the requested {size} of the profile owner with the specified{guid}.

http://social.yahooapis.com/v1/user/{guid}/profile/image/{size}

HTTP Operations Supported

  • GET

Query Parameters

  • callback
  • format

Scopes Required

  • GET:Read Yahoo! Profiles
