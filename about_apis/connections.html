Connections

Description

Connections returns the user's connections. A user's connections, which are mutually confirmed relationships, allow for a higher degree of shared information. The Connections API provides access to these confirmed associations.

For more information, see Connections in the Yahoo! Social API Reference.

URI

GET returns a collection of connections for the profile owner specified by {guid}.

http://social.yahooapis.com/v1/user/{guid}/profile

HTTP Operations Supported

  • GET

Query Parameters

  • callback
  • format
  • view

Matrix Parameters:

  • start
  • count

Scopes Required

  • GET:Read Yahoo! Profiles
