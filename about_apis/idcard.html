IDCard

Description

The IDcard contains the same core data as the Usercard with the exception of status information.

For more information, see IDCard Profile in the Yahoo! Social API Reference.

URI

GET returns the idcard profile of profile owner with the given {guid}.

http://social.yahooapis.com/v1/user/{guid}/profile/idcard

Filters

Calling GET on the following URI returns the idcard profiles of the users identified by GUIDs {guid1}, {guid2}, and {guid3}.

/v1/users.guid(144,976,541)/profile/idcard

HTTP Operations Supported

  • GET

Query Parameters

  • callback
  • format

Matrix Parameters:

  • start
  • count

Scopes Required

  • GET:Read (Shared) Yahoo! Profiles
