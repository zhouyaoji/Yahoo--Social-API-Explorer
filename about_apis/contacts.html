Contacts

Description

With the Contacts API, you can obtain and add to the a collection of resources that represent a user's contacts. To retrieve contacts of contacts you will need a separate operation and each contact owner's permission.

For more information, see Contacts in the Yahoo! Social API Reference.

URI

Use the following URI to obtain Contact Objects or add a Contact Object for a user identified by the {guid} value.

http://social.yahooapis.com/v1/user/{guid}/contacts

Filters

See Filtering Contacts.

HTTP Operations Supported

HTTP POST and PUT operations require permission from Yahoo!. You can make a request through the YDN Contact Form.

  • GET
  • POST
  • PUT

Query Parameters

  • format
  • view

Matrix Parameters:

  • start
  • count
  • bucket
  • maxbucketsize
  • minbucketcount
  • sort-fields
  • sort

See also: Filtering Contacts and Bucketing.

Scopes Required

  • GET:Read Yahoo! Profiles
  • POST:Write Yahoo! Profiles
  • PUT:Write Yahoo! Profiles
