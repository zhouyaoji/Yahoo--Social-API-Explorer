Usercard

Description

The Usercard is the core data in the user's profile containing slightly more information than the tinyusercard.

For more information, see Usercard Profile in the Yahoo! Social API Reference.

URI

GET returns the usercard profile of the profile owner with the given {guid}.

http://social.yahooapis.com/v1/user/{guid}/profile/usercard

Filters

Calling GET on the following URI returns the profiles of the users identified by GUIDs {guid1}, {guid2}, and {guid3}.

v1/users.guid(144,976,541)/profile/usercard

HTTP Operations Supported

  • GET

Query Parameters

  • callback
  • format

Matrix Parameters:

  • start
  • count

Scopes Required

  • GET:Read Yahoo! Profiles
