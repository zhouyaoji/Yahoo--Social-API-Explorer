Updates

Description

Updates are a stream of recent events and activites that help users stay in touch with their connections (friends). Examples of updates are posting to a blog, uploading photos, and adding users to the list of connections. Updates are generated by a wide variety of Yahoo! sites and Web applications. Typically, updates are displayed on the right side of a Web page, as on Yahoo! Profiles. With the Updates API, your applications can generate and retrieve updates stored by Yahoo!.

For more information, see Updates in the Yahoo! Social API Reference.

URI

You can use this URI to get a collection of the most recent updates generated by a user. The user is uniquely identified by the {guid} variable.

http://social.yahooapis.com/v1/user/{guid}/updates

Filters

Calling GET on the following URI returns the profiles of the users identified by GUIDs {guid1}, {guid2}, and {guid3}.
