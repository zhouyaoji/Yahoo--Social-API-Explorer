Tinyusercard for Contacts

Description

You can use the following syntax to obtain a tinyusercard for each of the contacts. The contacts?view=tinyusercard segment indicates that the objects returned will be annotated with nickname and image data. For a description of the tinyusercard, see Tinyusercard Profile.

For more information, see Tinyusercard for Contacts in the Yahoo! Social API Reference.

URI

GET returns the tinyusercards for contacts of the profile owner with the given {guid}.

http://social.yahooapis.com/v1/user/{guid}/contacts?view=tinyusercard
